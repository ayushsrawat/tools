# Developer Tools

A collection of utility scripts and CLI tools to automate daily tasks.

## Available Tools

| Tool | Description | Usage |
| :--- | :--- | :--- |
| **thumbnailer.py** | Generates high-quality square thumbnails for images. Features smart "top-center" cropping to preserve faces in portraits and bulk directory processing. | `uv run thumbnailer.py <path> --size 256` |
